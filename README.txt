1.0 Overview
================================================================================
C++ and Python implementation of a brute force algorithm that iterates through
all possible values for strings of varying lengths. Worked well for BetaFour
challenge in BCTF 2016.

2.0 Build and Test
================================================================================

To build:

	Using GNU C++:
		make brutiful

	From a Windows SDK Prompt or Visual Studio Tools prompt:
		cl brutiful.cpp

To test:
	Run:
		brutiful 4
	or:
		python brutiful.py
	and it will converge on a solution very quickly.
